# 간단한 md파일 작성

## 제목/부제목
```
# 제목
## 제목
### 제목
#### 제목
##### 제목
###### 제목
```
### 적용 결과

# 제목
## 제목
### 제목
#### 제목
##### 제목
###### 제목

## 글씨 기울임
```
  *기울임1*
  _기울임2_
```
### 적용 결과
*기울임1*  
_기울임2_

## 굵은 글씨
```
  **굵게**
  __굵게__
```

### 적용 결과
**굵게**  
__굵게__

## 취소선
```
  ~~취소선~~
```
### 적용 결과
~~취소선~~

## 인용글
```
> 1
> > 2
```
### 적용 결과
> 1
>> 2

## 글 머리기호(+,-,*)
```

  +2
```
### 적용 결과
